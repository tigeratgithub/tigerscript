### 设置拥塞控制算法为bbr
```shell
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/tigeratgithub/tigerscript/main/enable_bbr.sh)"
```
或者 简单版本
```shell
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/tigeratgithub/tigerscript/main/simple_bbr.sh)"
```
